# 渐进式交付说明

## M0 骨架可运行

- 页面主结构可见
- 组件骨架联通

## M1 核心交互闭环

- 输入、播放、代码同步可用
- 核心快捷键和拖拽可用

## M2 信息密度增强

- 箭头、标签、状态迁移完整
- 空节点与数据流可见

## M3 多解法对比

- 多解法隔离
- 切换重置正确

## M4 发布封版

- CI 全绿
- Pages 可访问
- 验收报告完整

## 阶段与最终通过关系（强制）

- 阶段通过（M0-M4）仅代表能力达成度，不代表最终可发布。
- 即使达到 M4，仍必须通过质量门禁 B1-B7 才能判定 PASS。
- 若任一 Gate 失败，结论为 FAIL，必须进入失败闭环继续修复。

## 建议

- 普通题至少做到 M2
- 教学重点题建议做到 M4
