# Playwright 用例清单（E2E-ID）

> 执行基线：Chromium + 1920x1080 + `trace: retain-on-failure`

## 必过集合（Blocking E2E）

发布前必须 100% 通过：

- E2E-001 首屏单屏布局与关键元素可见
- E2E-002 顶部三类外链新标签行为
- E2E-003 输入合法性（非法不重算、合法重算）
- E2E-005 语言切换与刷新记忆
- E2E-007 快捷键 Left/Right/Space/R
- E2E-008 进度条拖拽跳步
- E2E-009 画布缩放与平移
- E2E-014 GitHub 徽标提示与 Star 数展示
- 若题目含多解法：E2E-011 也必须通过

## 扩展集合（建议全部执行）

- E2E-004 样例与随机输入可用
- E2E-006 代码高亮与变量旁注联动
- E2E-010 算法思路弹窗开关
- E2E-012 交流群浮层与二维码比例
- E2E-013 速度偏好刷新后保留
- E2E-015 无紫色主题 token

## 建议输出

- 每条失败用例保留 trace/video/screenshot
- 验收报告中记录：E2E-ID、失败原因、修复 commit、复测结果
- 任何 Blocking E2E 失败都必须走失败闭环后再复测
