# 验收评分量表（Rubric）

## 评分方式

每项 0-5 分：

- 0: 缺失
- 1: 很差
- 2: 可运行但不稳定
- 3: 达标
- 4: 良好
- 5: 优秀

## 评分项

| 项目 | 描述 | 权重 |
| --- | --- | --- |
| 首屏信息架构 | 关键模块是否首屏完整可用 | 10% |
| 分镜清晰度 | 步骤是否足够细且可理解 | 15% |
| 代码同步准确性 | 高亮与变量值是否准确 | 15% |
| 画布信息密度 | 是否传达有效信息而非装饰 | 15% |
| 交互可控性 | 快捷键、拖拽、切换是否顺畅 | 10% |
| 多解法可比性 | 解法隔离是否清晰可比较 | 10% |
| 稳定性 | 异常输入/边界数据下是否稳定 | 10% |
| 工程发布质量 | CI 与部署可用性 | 10% |
| 文档完整性 | README/测试/验收文档是否齐全 | 5% |

## 通过标准

- 综合分 >= 80
- 且无 P0/P1 问题
- 且一票否决项为 0
